# roboguard-pi
This repo contains the packages required to ron on roboguard-pi.

## Dependencies
- Docker (optional for development)
Or
- vcs
- ros humble

## Setup

### Docker
Install docker:
```sh
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh ./get-docker.sh
```

### Development
Import repos:
```sh
vcs import src < roboguard.repos
```
